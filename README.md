# StudyPal

Your friendly neighborhood study buddy.

## Description

StudyPal is a CLI agent designed to assist with software engineering tasks. It helps users safely and efficiently by adhering to project conventions and best practices.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/studypal.git
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```

## Usage

To start the development server, run:

```bash
npm run dev
```

To build the project, run:

```bash
npm run build
```

To start the project in production, run:

```bash
npm run start
```

## Scripts

- `npm run test`: Runs the test suite.
- `npm run dev`: Starts the development server.
- `npm run build`: Builds the project for production.
- `npm run start`: Starts the production server.
